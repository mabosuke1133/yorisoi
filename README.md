# Yorisoi（寄り添い）

## サイト概要

### サイトテーマ
「Yorisoi」は、業務知識の共有と心の状態の可視化を通じて、離職防止を支援する社内コミュニケーションSNSサイトです。

### テーマを選んだ理由
　マネージャーとして従事しているときに離職率が高い時期があり、「相談したいと自分から言いづらい」「不安解消されないことで日に日に孤独感が増している」が主な離職理由でありました。特に若手は社内で気を遣うこともあり、余計に気疲れしモチベーションが下がるため常にアンテナを張る必要がありますが、対面にて1on1しようにも「言いたいことをしっかり伝えることができるのか」といった準備が無ければ面談時間も長くなりますし、結局解決できなければ両者ともに精神的疲労が大きくなります。また、業務のコツや気づきを気軽にアウトプットできる場がないのも問題かと思いました。そこで離職率を減らせるよう、いち早く社員の状況を把握できるツールがあれば管理側もよりよい対応ができるのではないかと考えました。今の時代リモートも普及しており常にデータを通じて社員の心の変化に気づくことでリモート社員の助けにもなるかと思い、このテーマにしました。

### ターゲットユーザー
- **一般ユーザー（若手・一般社員）**：自分の頑張りを見てほしい、気軽に相談したい、役立つ情報を得たい人。
- **管理者ユーザー（上司・メンター）**：社員の状態を把握したい、離職のサインを早めに察知したい人。

### 主な利用シーン

- **業務に対して孤独感を解消したい人** ：業務の相談を気軽に投稿し、先輩や同僚が「いいね」やコメントをすることで自身の貢献を可視化したい時
- **過去投稿をナレッジとして検索したい人** ：過去に誰がどんな壁にぶつかり、どう解決したかを知りたい時
- **心のコンディションを報告したい人** ：直接口では言いにくい「気分が落ちている」ことをシステムを通じて報告したい時
- **社員の不調を早期に汲み取りたい管理者** ：数日間「気分が落ちている」が続いている社員を自動でピックアップし、手遅れになる前のきっかけ作りしたい時
